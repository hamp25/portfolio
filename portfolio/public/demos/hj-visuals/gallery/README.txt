HOW TO ADD YOUR REAL PHOTOS
============================

1. Drop your photo files directly into this folder (public/gallery/).

2. Name them to match the list in src/lib/businessInfo.ts, for example:
   event-01.jpg
   portrait-01.jpg

   (Or rename the entries in businessInfo.ts to match your own filenames —
   either direction works, just make sure the names match.)

3. Recommended: JPG, around 1600px wide, under 500KB each, so the site
   stays fast to load.

4. That's it — no code changes needed. Refresh the site and your real
   photos will replace the elegant placeholder frames automatically.

If a photo file is missing, the site won't break — it will just keep
showing the placeholder frame for that slot until you add the image.
